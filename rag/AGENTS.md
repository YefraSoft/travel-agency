# RAG — Agencia de Viajes

Agente conversacional para WhatsApp con Python, LlamaIndex, ChromaDB y Ollama.

## Reglas

- El RAG nunca procesa pagos.
- Si no puede resolver una consulta, debe escalar al agente humano.
- Debe consultar datos reales del backend cuando corresponda.
- No registrar informacion sensible completa en logs.
